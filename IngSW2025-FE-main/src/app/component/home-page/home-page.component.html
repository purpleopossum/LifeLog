Small habits. Big changes.

Building lasting habits isn't about willpower — it's about the system you create. This app helps you design tiny, repeatable actions, track them every day, and turn them into the routines that shape your life.

How it works

  1. Create a habit, pick when it belongs in your day (morning / afternoon / evening) and set an easy starting action.
  2. Check in every day with a quick status and optional note & mood.
  3. Reflect & improve — review notes and streaks, then make the habit easier, more obvious, or more rewarding.

The 4 Keys (from "Atomic Habits")

1

Make it obvious

Design cues and triggers so the right choice is visible and automatic.

2

Make it attractive

Bundle habits with things you enjoy to increase desire and consistency.

3

Make it easy

Start tiny — small wins stack into long-term progress.

4

Make it satisfying

Give immediate feedback or rewards so the habit is reinforced.
